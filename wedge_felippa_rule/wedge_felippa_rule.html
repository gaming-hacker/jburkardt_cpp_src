WEDGE_FELIPPA_RULE
Quadrature Rules for the Wedge

WEDGE_FELIPPA_RULE, a C++ code which generates quadrature rules to estimate integrals over the interior of the unit wedge in 3D.

The interior of the unit wedge in 3D is defined by the constraints: 

        0 <= X
        0 <= Y
             X + Y <= 1
       -1 <= Z <= +1
