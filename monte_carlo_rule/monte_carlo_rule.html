MONTE_CARLO_RULE
Monte Carlo Sampling Regarded as Quadrature Rule

MONTE_CARLO_RULE, a C++ program which generates a dataset of N random M-dimensional points, regards it as a quadrature rule for the unit hypercube, and writes out three files of information.

Usage:

monte_carlo_rule m n s
where
  • m is the spatial dimension;
  • n is the number of points to generate;
  • s is the initial seed value.

The data is written to three files:

  • an "R" or "region" file, which lists two points that bound the region, which will be the unit hypercube.
  • a "W" or "weight" file, which lists the weight for each abscissa. each weight will equal 1/N.
  • an "X" or "abscissa" file, which lists the abscissas of the rule. These will simply be the N random points in the M-dimensional hypercube.

Related Data and Programs:

LINE_FELIPPA_RULE, a C++ library which returns the points and weights of a Felippa quadrature rule over the interior of a line segment in 1D.

LINE_NCO_RULE, a C++ library which computes a Newton Cotes Open (NCO) quadrature rule, using equally spaced points, over the interior of a line segment in 1D.

monte_carlo_rule_test

POWER_RULE, a C++ program which constructs a power rule, that is, a product quadrature rule from identical 1D factor rules.

QUADRATURE_RULES_UNIFORM, a dataset directory which contains quadrature rules for M-dimensional unit cubes, based on a uniform pseudorandom sequence. stored as a file of abscissas, a file of weights, and a file of region limits.
