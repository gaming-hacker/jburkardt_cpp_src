pwl_interp_2d_test

pwl_interp_2d_test, a C++ code which calls pwl_interp_2d(), which evaluates a piecewise linear interpolant of data depending on a 2D argument, defined on on a product grid, so that p(x(i),y(j)) = z(i,j).
