fd1d_advection_ftcs_test

fd1d_advection_ftcs_test, a C++ program which calls fd1d_advection_ftcs() to apply the finite difference method to solve the time-dependent advection equation ut = - c * ux in one spatial dimension, with a constant velocity, using the FTCS method, forward time difference, centered space difference, writing graphics files for processing by gnuplot.

Related Data and Programs:

fd1d_advection_ftcs, a C++ program which applies the finite difference method (FDM) to solve the time-dependent advection equation ut = - c * ux in one spatial dimension, with a constant velocity, using the forward time, centered space (FTCS) difference method, writing graphics files for processing by gnuplot.

Source Code:

  • fd1d_advection_ftcs_test.sh, runs all the tests.
  • fd1d_advection_ftcs_test.txt, the output file.

Graphical output for this program was created using GNUPLOT. Data at selected time steps was written to a "data" file, and the appropriate GNUPLOT commands were written to a "command" file. The plot can be created by the command 

        gnuplot < advection_commands.txt
  • advection_data.txt, the solution data.
  • advection_commands.txt, gnuplot commands to plot the data.
  • advection.png, a (not very satisfactory) image of the solution.
