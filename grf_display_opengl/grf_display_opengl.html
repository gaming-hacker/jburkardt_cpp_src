GRF_DISPLAY_OPENGL
Display a GRF Graph Using OpenGL

GRF_DISPLAY_OPENGL, a C++ program which reads a GRF file containing information about an abstract graph and displays a plot using OPENGL.

Usage:

grf_display_opengl prefix
where
  • prefix.grf is the GRF file.
