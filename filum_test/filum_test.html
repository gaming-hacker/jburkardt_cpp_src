filum_test

filum_test, a C++ program which calls filum(), which handles information in text files.

filum, a C++ library which can work with information in text files.

DATA_***.TXT is a short sequence of three files, each having 8 data items, 3 on the first line, 3 on the second, and 2 on the third line, to be examined by FILE_SEQUENCE_SIZE.

  • data_100.txt, the first file in a sequence of 3.
  • data_101.txt, the second file in a sequence of 3.
  • data_102.txt, the third file in a sequence of 3.
