MANDELBROT_PPM
Binary PPM Image of the Mandelbrot Set

MANDELBROT_PPM, a C++ program which creates a binary PPM image of the Mandelbrot set, by Eric Weeks.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

MANDELBROT_PPM is available in a C version and a C++ version.

Related Data and Programs:

MANDELBROT, a C++ program which generates an ASCII PPM image of the Mandelbrot fractal set;

MANDELBROT_OPENMP, a C++ program which generates an ASCII Portable Pixel Map (PPM) image of the Mandelbrot fractal set, using OpenMP for parallel execution.

mandelbrot_ppm_test

PPMB_IO, a C++ library which handles the Portable Pixel Map (binary) format.
