fem2d_heat_rectangle_test

fem2d_heat_rectangle_test, a C++ program which calls fem2d_heat_rectangle(), which solves the time-dependent 2D heat equation using the finite element method in space, and a method of lines in time with the backward Euler approximation for the time derivative.

Related Data and Programs:

fem2d_heat_rectangle, a C++ program which uses the finite element method (FEM) to solve the 2D time dependent heat equation on the unit square, using a uniform grid of triangular elements.
