wedge_grid_test

wedge_grid_test a C++ code which calls wedge_grid(), which generates a grid of points over the interior of the unit wedge in 3D.

The test program creates some files for use by GNUPLOT.

  • wedge_commands.txt, commands for GNUPLOT.
  • wedge_vertices.txt, the vertices of the wedge.
  • wedge_nodes.txt, the grid points.
  • wedge.png, a plot of the grid.
