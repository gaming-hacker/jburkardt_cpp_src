catalog_test

catalog_test, a C++ code which calls catalog(), which reads a C, C++, FORTRAN77, FORTRAN90, MATLAB, Python, or R code and prints every line that begins with a special index tag. If the code has been marked up expecting this convention, a handy table of contents will be created.
