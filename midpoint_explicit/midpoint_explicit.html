midpoint_explicit

midpoint_explicit, a C++ code which solves one or more ordinary differential equations (ODE) using the (explicit) midpoint method, also known as the modified Euler method.

Languages:

midpoint_explicit is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

midpoint, a C++ code which solves one or more ordinary differential equations (ODE) using the (implicit) midpoint method.

predator_prey_ode, a C++ code which solves a pair of predator prey ordinary differential equations (ODE).

stiff_ode, a C++ code which considers an ordinary differential equation (ODE) which is an example of a stiff ODE.
