discrete_pdf_sample_2d_test

discrete_pdf_sample_2d_test, a C++ program which calls discrete_pdf_sample_2d() to demonstrate how to construct a Probability Density Function (PDF) from a table of sample data over a 2D region, and then to use that PDF to create new samples.
