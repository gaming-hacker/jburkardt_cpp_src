tiler_3d_test

tiler_3d_test, a C++ code which calls tiler_3d(), which carries out transfinite interpolation to "tile" a 3D box that has already been subdivided into sub-boxes.

Related Data and Programs:

tiler_3d, a C++ code which demonstrates a 3D example of transfinite interpolation.
