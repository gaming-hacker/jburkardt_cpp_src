triangulation_node_to_element_test

triangulation_node_to_element_test, a C++ code which calls triangulation_node_to_element(), which reads datafiles describing a set of nodes, their triangulation, and the value of one or more quantities at each node, and outputs a file that averages the quantities for each element. This operation in effect creates an "order1" finite element model of the data.
