triangulation_t3_to_t4_test

triangulation_t3_to_t4_test, a C++ code which calls triangulation_t3_to_t4(), which reads information describing a triangulation of a set of points using 3-node "T3" triangles, and creates a 4-node "T4" triangulation.

Related Data and Programs:

triangulation_t3_to_t4, a C++ code which reads information about a 3-node triangulation and creates data defining a corresponding 4-node triangulation (vertices + centroid);

Source Code:

  • triangulation_t3_to_t4_test.sh, runs all the tests.
  • triangulation_t3_to_t4_test.txt, the output file.

EXAMPLE is a simple set of example data:

  • example_nodes.txt, the nodes for the 3-node triangulation.
  • example_nodes.png, a PNG image of the nodes.
  • example_elements.txt, the triangles for the 3-node triangulation.
  • example.png, a PNG image of the elements.
  • example_t4_nodes.txt, the nodes for the 4-node triangulation.
  • example_t4_noes.png, a PNG image of the nodes.
  • example_t4_elements.txt, the triangles for the 4-node triangulation.
  • examples_t4.png, a PNG image of the elements.
