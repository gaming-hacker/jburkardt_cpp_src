tet_mesh_boundary_test

tet_mesh_boundary_test, a C++ code which calls tet_mesh_boundary(), which reads information defining a tetrahedral mesh of points in 3D, and determines the triangular faces that form the boundary of the mesh; it writes out files containing the nodes and elements defining this "TRI_SURFACE".

Related Programs:

tet_mesh_boundary, a C++ code which returns the nodes and faces of the boundary of a tetrahedral mesh, which themselves form a 3D triangular mesh or tri_surface.
