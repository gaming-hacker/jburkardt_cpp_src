fem1d_lagrange_test

fem1d_lagrange_test, a C++ program which calls fem1d_lagrange(), which sets up the matrices and vectors associated with the finite element method (FEM) solution of a boundary value problem (BVP) -u''+u=f(x), using Lagrange basis polynomials.
